# 本书简介
Web开发领域知识庞杂，各类库、标准、框架等知识盘根错节，像一棵百年老树的树根。新手朋友面对它们如同面对一个刺猬，无从下手。
本书将从GoFrame框架出发，带领新手朋友们实际开发一个项目：星辰英语本。旨在为读者提供一份深入浅出的GoFrame框架学习指南，帮助读者快速掌握GoFrame框架。
以实际项目为导向，分享更多专业、实用的编程技巧和经验，让您学有所成是作者的期望！
